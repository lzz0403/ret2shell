比赛事件目前有两种，分别是 **冻结** 和 **解冻**。

## 消息格式

你收到的 JSON 消息如下：

```json
{
  "game": {
    "event_type": "freeze", // "freeze" | "unfreeze"，保证不会有其他值
    "operator": {
      "id": 1,
      "account": "admin",
      "nickname": "管理员",
      ... // 其他数据库字段，不保证完整，也不保证今后的版本不变更，即使存在也不要使用这些字段
    }
  }
}
```

## 触发规则

管理员更新比赛状态时，如果更改了比赛状态为 **冻结** 或 **解冻**，则会立即触发该事件。

## 使用场景

适合在比赛通知中使用，例如使用机器人在群中自动通知比赛状态的变化，以便于选手合理安排自己的作息。
